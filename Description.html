Data Description

The dataset represents lottery draw results recorded from 2010 to 2025, showing how frequently each ball number was drawn and when it was last selected. Each entry includes three key variables: Ball Number, Times Drawn, and Last Drawn Date. The Ball Number identifies the specific lottery ball, Times Drawn indicates how often that number appeared during a given period, and Last Drawn Date records the most recent time it was drawn. In the visualization, a scatter plot is used to display this information clearly.

Why This Dataset Is Interesting

This dataset is particularly engaging because it connects mathematical probability with real-world data. Many people associate the lottery with luck, yet rarely consider how randomness behaves statistically. By examining the frequency of drawn numbers, we can observe whether true randomness produces an even spread or if some numbers appear slightly more often by coincidence.
